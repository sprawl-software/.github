**Sprawl** is an editor-agnostic configuration management engine for AI coding tools.  
One config. Every editor. Version-controlled in Git.
- 🔧 Works with Cursor, VS Code, Zed, Vim, Aider
- 📦 Zero external dependencies — stdlib-only Python
- 🔒 Workspace sandboxing — scope-locked file servers per project
- ⚡ `sprawl init` → 30 seconds to full team config
```bash
pip install sprawl-cli
```
