## Who We Are

Sprawl.software is the **Agentic Containment & Telemetry Platform** for autonomous software engineering.

AI coding agents are changing how software is built. But they operate in a black box — scanning folders, running shell commands, and making API calls without oversight. Engineering leads have no visibility into what code the model reads, how much it costs, or whether corporate IP is leaking.

We build the tools, the telemetry runtime, and the governance layer that make autonomous AI development **safe, predictable, and audit-ready**.

## What We Build

| | Capability | What It Does |
|---|---|---|
| 🔧 | **Developer CLI** | Zero-dependency workspace sandboxing, universal IDE binding, and GitOps configuration sync. Write rules once, compile everywhere. |
| 🔒 | **MCP Security Servers** | Directory-locked filesystem guards that block path traversal in <2ms. Agents cannot escape the project boundary. |
| 📊 | **Telemetry & Observability** | Out-of-band session database scraping. Context coverage auditing. Token spend tracking. All behind your firewall. |

| 📋 | **Project Management (MDPM)** | Local-first Markdown Project Management. Zero-dependency Kanban boards, milestone scaffolding, and pipeline automation. |
| 📐 | **Open Standards (AFF)** | [Atomic Agentic Fabric](https://github.com/w3bwizart/atomic-agentic-fabric-specification) — the open-source specification for structuring portable agent capabilities. |

## How We Build

- **100% Local** — Zero cloud dependency. Zero telemetry. Zero network calls.
- **Zero Dependencies** — Pure Python standard library. A senior engineer can audit the full codebase in 48 hours.
- **Standard Library Only** — No third-party supply chain risk. Sub-100ms boot time.
- **MCP-Native** — Model Context Protocol over stdio for agent filesystem governance.
- **BSL-1.1 Licensed** — Open code, protected distribution. Free for non-production use.
